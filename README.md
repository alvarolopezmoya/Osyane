# Osyane — Sistema de Gamificación FISEI-UTA

Sistema de gamificación y ranking académico para estudiantes de Ingeniería en Software · Universidad Técnica de Ambato.

## Características

- 📊 **Dashboard** — XP, nivel, progreso, actividad reciente
- 🏆 **Ranking** — Tabla de líderes con podio
- 🏅 **Insignias** — Badges ganados y por desbloquear
- 📈 **Progreso** — Gráficas de XP y avance por asignatura
- 👩‍🏫 **Docente** — Panel para otorgar XP e insignias

## Tecnologías

- React 18 + Babel (sin bundler)
- Tailwind CSS v4 (browser)
- Recharts para gráficas
- PWA con manifest.json

## Cómo correr localmente

Abre `index.html` directamente en el navegador, o usa un servidor local:

```bash
npx serve .
```

## Despliegue

Arrastra la carpeta a [netlify.com/drop](https://app.netlify.com/drop) para publicar en segundos.
